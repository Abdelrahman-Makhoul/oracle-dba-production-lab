# Oracle DBA Production Lab

This repository documents my hands-on learning journey in Oracle Database Administration using Oracle Database 19c and Oracle Linux.

## Lab Environment

- OS: Oracle Linux Server 8.10
- Database: Oracle Database 19c Enterprise Edition
- Database Name: ORCL
- SID: orcl
- PDB: ORCLPDB
- Archive Log Mode: Enabled
- Listener Port: 1521
- Lab Type: Production-like home lab

## Learning Goals

- Understand Oracle Database architecture
- Practice daily DBA tasks
- Work with RMAN backup and recovery
- Monitor database health
- Troubleshoot common Oracle issues
- Automate DBA checks using shell scripting

## Completed Phases

### Phase 1: Oracle Foundations

Topics covered:

- Database vs Instance
- SGA and Oracle memory structure
- Background processes: PMON, SMON, DBWR, LGWR, CKPT
- Datafiles, Control Files, and Redo Logs
- Tablespaces, Segments, Extents, and Blocks
- Alert Log, Trace Files, and ADRCI
